# Form builder

Generator for forms by users with some simple controls.

## Getting Started
### Prerequisites

Install yarn from https://yarnpkg.com/en/docs/getting-started

### Installing

Run your bash and get to the project directory.

Run: yarn install

## Running
Run yarn run start
